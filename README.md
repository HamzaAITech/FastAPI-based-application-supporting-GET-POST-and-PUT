FastAPI 

This project API built using FastAPI.
It demonstrates CRUD operations, request validation, and dynamic path parameters.

📌 Features

Root endpoint (/)

Dynamic greeting using path parameters (/Hello/{name})

Create Todo items (POST /todos)

Retrieve all Todos (GET /todos)

Retrieve single Todo by ID

Update existing Todo (PUT /todos/{todo_id})

Request validation using Pydantic models

Interactive API documentation using Swagger UI
